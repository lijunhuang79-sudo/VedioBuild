# 数学故事《时钟小镇的快递任务》— 生成小视频说明

本说明提供两种方式，用 VedioBuild 生成涵盖演讲全文要点的小视频。

---

## 方式一：在 VedioBuild 网页直接使用（推荐）

在 **AI 视频工厂** 的「视频主题」框里，**整段复制**下面这段主题，然后点击「AI 生成视频」即可。系统会根据主题生成 6 镜故事画面与旁白，尽量覆盖以下要点：时钟小镇、大宝小帅、31 颗糖分 4 盒、有余数除法 7 余 3、分针从 3 走到 5 共 10 分钟、1000+80+5=1085 颗糖、数学在身边。

### 视频主题（复制到「视频主题」框）

```
数学故事《时钟小镇的快递任务》：在神奇的时钟小镇，小快递员大宝和跑得飞快的好朋友小帅接到镇长任务——10分钟内把31颗星星糖果送到森林邮局。大宝只有4个小盒子，小帅用有余数的除法：31除以4等于7余3，每盒装7颗还剩3颗。镇长说分针从数字3走到数字5必须送到；大宝看钟表：1大格5分钟，从3到5走2大格，2乘5等于10分钟，时间刚刚好。大宝准时送到后，邮局阿姨问：1000颗奶糖、80颗水果糖、5颗软糖一共多少？大宝答1000加80加5等于1085颗，这就是万以内的数。数学就在我们身边：时分秒、有余数的除法、万以内的数能解决好多问题。一个用数学完成任务的小快递员故事。
```

**使用步骤：**

1. 本地：先启动后端与 Worker（如 `./scripts/start-backend.sh`），再打开前端 dashboard。
2. 在「视频主题」中粘贴上面整段主题。
3. 可选：选择「大片风格」（如默认或科技）、配音音色、BGM。
4. 点击「AI 生成视频」，等待生成完成后下载。

生成出的视频会是 6 个镜头，旁白约 80～120 字（由 AI 从主题浓缩），画面会尽量贴合：小镇与快递员、分糖与 7 余 3、钟表分针、邮局与糖果、数学在身边等情节。

---

## 方式二：在应用内使用「自定义旁白」（推荐，涵盖全部信息）

在 **AI 视频工厂** 的 dashboard 中已支持**自定义旁白**模式：

1. 点击 **「自定义旁白」** 切换按钮（与「主题生成」并列）。
2. **视频标题**填：`时钟小镇的快递任务`（或任意简短标题）。
3. 在 **旁白全文** 中粘贴下面「旁白全文」。
4. 在 **六镜场景描述** 中依次粘贴下面 6 条场景描述（每条对应一镜）。
5. 选择配音、风格、BGM 后点击「AI 生成视频」。

生成出的视频将按你写的旁白一字不差配音，画面按六镜描述生成，可完整涵盖演讲所有信息。

### 旁白全文（约 6 段，对应 6 镜）

```
在很远的地方有一座神奇的时钟小镇，镇上住着快递员大宝和跑得飞快的好朋友小帅。镇长交给大宝一个任务：10分钟内把31颗星星糖果送到森林邮局。
大宝只有4个小盒子，小帅说用有余数的除法——31除以4等于7，还剩下3颗，每盒装7颗，多出3颗。
分针从数字3走到数字5必须送到；分针走1大格是5分钟，从3到5走2大格，2乘5等于10分钟，时间刚刚好。
大宝一路快跑，准时把糖果送到了邮局。
邮局阿姨问：1000颗奶糖、80颗水果糖、5颗软糖合起来多少？大宝答：1000加80加5等于1085颗，这就是万以内的数。
数学就在我们身边：时分秒、有余数的除法、万以内的数，能解决好多好多问题。
```

### 六镜场景描述（用于每镜背景图）

| 镜头 | 场景描述 |
|------|----------|
| 第1镜 | 童话风时钟小镇，街道与钟楼，小快递员与好朋友在镇口 |
| 第2镜 | 31颗星星糖果与4个小盒子，桌面分装，有余数除法的画面感 |
| 第3镜 | 大钟表特写，分针从3指向5，强调时间与分钟 |
| 第4镜 | 森林小路，快递员奔跑，远处邮局建筑 |
| 第5镜 | 邮局室内，阿姨与糖果：奶糖、水果糖、软糖，数字1085 |
| 第6镜 | 温暖教室或书房，时钟、算式与书本，数学在身边的总结画面 |

把上述「旁白全文」和「六镜场景描述」按系统要求的格式（如 JSON 或表单）传入，即可生成涵盖演讲全部要点的小视频。

---

## 小结

- **快速出一版**：用方式一，在「主题生成」下把主题复制到「视频主题」即可。
- **一字不差覆盖全文**：用方式二，在 dashboard 切换到「自定义旁白」，粘贴旁白全文与六镜场景描述后生成。
